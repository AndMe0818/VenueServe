# AipServer

#### 介绍
 基于大学体育场研究的体育馆场地预约登记管理系统的后台接口

#### Tips:

- 这个系统也是我自己尝试着从零开始搭建，其中肯定也有部分错误，请大佬们轻喷。

- 其中的数据库可能缺少一些脚本比较难跑起来


#### 软件架构
软件架构说明：
B/S架构
使用了 node express mongoDB 以及一些npm 工具 可以对用户信息、场地信息、预约信息进行管理。

#### 安装教程
需要有node环境以及mongoDB数据库
1.  npm install
2.  nodemon app.js  / node app.js
3.  xxxx

#### 使用说明

1.  用户信息管理
2.  用户预约
3.  场地信息
